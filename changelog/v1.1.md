Changelog NEAT v1.1
===================

This release covers the merged pull requests after the `v1.0-post` tag. The
main focus is broader simulator compatibility, improved export workflows, and
targeted fixes in morphology handling and compartment fitting.

Highlights
==========
* Added compatibility with NEURON 9 and CoreNEURON, including more robust model
  installation and loading across NEURON versions, updated mod files, and CI
  coverage for the CoreNEURON backend (`#194`).
* Added Brian2 export support for compartmental models, including concentration
  mechanisms and accompanying tests (`#194`).
* Added new x-axis color schemes for unravelled morphology visualizations
  (`#187`).

Bug fixes
=========
* Guard against negative conductances provided to `PhysTree` (`#171`).
* Fixed a singular-matrix failure in leak current potential fitting
  (`#189`).

Maintenance
===========
* Merged the `bugfixes` branch after the targeted fixes above. This umbrella PR
  does not add separate user-facing changes beyond the merged fixes already
  listed here (`#195`).
* PR `#166` was closed without merge and does not contribute release changes.
